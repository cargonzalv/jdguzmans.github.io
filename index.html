Projects

Repositorio de Estudios Geotécnicos

Academical project directed by professor Jaime Wills, co-developed with Diego Rodriguez Baquero.
It is basically a web repository of colombian soil samples; users can upload information about the geomechanical properties of a stratigraphy and they can even upload 'pdf' files of them. Check it out!
Main technologies used: NodeJS, ExpressJS, MariaDB and Sequelize.

Leasy

Web application that helps colombian civil contractors with their experience accreditation in order to get the best scores in the public biddings. Would you like to try it? Enter witht the username 'abc' and password 'abc'.
Main technologies used: NodeJS, ExpressJS, MongoDB, Mongoose, RabbitMQ and Java.

I like to bragg about this project because in the background it does a really heavy algorithm that tryes all the combinations of contracts with the user's and its partners' contracts. This processing could not be done in the back-end of the app because a single client could block the system, so it uses a message queue, which enables asynchronous comunication with a 'worker' server pool that take the messages, execute the algorithm and send the results via email.
